# 🎵 Harmusic

Harmusic é um aplicativo desktop desenvolvido em Python utilizando CustomTkinter e a API da Deezer.

O objetivo do projeto é permitir a busca de artistas musicais, exibindo informações do artista e suas músicas mais populares de forma moderna e intuitiva.

---

# 🚀 Tecnologias Utilizadas

- Python 3.13
- CustomTkinter
- Requests
- Pillow (PIL)
- Deezer API

---

# 📸 Funcionalidades

✅ Buscar artistas pelo nome  
✅ Exibir foto do artista  
✅ Mostrar quantidade de fãs  
✅ Abrir perfil do artista no Deezer  
✅ Exibir Top músicas do artista  
✅ Pesquisa utilizando tecla ENTER  
✅ Interface moderna utilizando CustomTkinter
